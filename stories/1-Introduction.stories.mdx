### Conventions ->

- All assets goes to `/public/assets`.
- mobile first approach [link](https://developer.mozilla.org/en-US/docs/Web/Progressive_web_apps/Responsive/Mobile_first).
- Keep stories in the same folder of the component.
- Template for components is here -> `src/Template`.
- Adding new Icons:
  - copy the svg from figma as svg.
  - use this website to convert it to a React Component [link](https://svg2jsx.com/).
  - create a new export in `src/Icons/Icons.jsx`.
